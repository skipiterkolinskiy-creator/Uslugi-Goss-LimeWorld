# УслугиГосс v2

## Главное обновление

Добавлены лицензии с тестами:

- автомобиль: 13 из 15;
- грузовик: 13 из 15, используется общий дорожный банк вопросов;
- мотоцикл: 13 из 15;
- оружие: 14 из 16;
- рыбалка: без теста, только пошлина;
- охота: доступна только после получения лицензии на оружие, затем пошлина.

После успешного теста пользователь оплачивает пошлину через `/donate` и отправляет скрин. Заявка поступает в административный чат.

## Административная панель

Команда `/panel` работает только:

1. В чате, указанном в `ADMIN_CHAT_ID`.
2. Для Telegram ID, перечисленных в `ADMIN_IDS`.

Панель позволяет:

- искать персонажей;
- просматривать заявки;
- редактировать любое поле персонажа;
- выдавать штраф;
- объявлять в розыск;
- снимать розыск;
- отмечать персонажа живым или мёртвым.

## Две таблицы

- `GOS_SPREADSHEET_ID` хранит полную базу УслугиГосс.
- `MVD_SPREADSHEET_ID` остаётся старой базой КПК МВД.

После одобрения паспорта бот:

1. Создаёт полную запись в таблице УслугиГосс.
2. Добавляет сокращённую запись в старую базу МВД.

Код входа, Telegram ID и дата регистрации в базу МВД не передаются.
Username передаётся, только если в старой таблице существует столбец `Username`.

## Перед установкой

Положите рядом с `bot.py` файл `credentials.json`.
Сервисному аккаунту должны быть выданы права редактора на обе Google Таблицы.

## Запуск

```bash
pip install -r requirements.txt
python bot.py
```
