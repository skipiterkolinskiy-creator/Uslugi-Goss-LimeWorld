# УслугиГосс v1

Реализовано:

- регистрация персонажа;
- паспорт только после одобрения в `ГОСУСЛУГИ АДМ ЧАТ`;
- автоматический следующий номер паспорта;
- код входа после выдачи паспорта;
- возраст 14–17: военный билет не предусмотрен;
- возраст 18–200: вопрос о наличии военного билета;
- фото для документов;
- медкарта, которую заполняет игрок и подтверждает администратор;
- одобрение и отклонение с причиной;
- уведомление пользователя о решении;
- кнопки `Мёртв` и `Жив`;
- общая Google Таблица с КПК МВД;
- отображение штрафов, розыска и лицензий;
- предупреждение: поддельный скрин оплаты = x2 штраф + розыск;
- логи действий администраторов.

## Настройка

1. Скопируйте `.env.example` в `.env`.
2. Укажите токен нового бота, ID админ-чата и ID модераторов.
3. Положите рядом `credentials.json`.
4. Дайте сервисному аккаунту доступ редактора к таблице.
5. Запуск:

```bash
pip install -r requirements.txt
python bot.py
```

Модераторы указываются через запятую:

```env
ADMIN_IDS=8548608434,123456789,987654321
```
